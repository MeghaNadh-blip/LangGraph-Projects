# LangGraph AI Chatbot 🚀

A simple AI chatbot project built using **LangGraph**, **LangChain**, and **Groq LLM**.

## 📌 Features

- AI Chatbot using Groq
- LangGraph workflow
- Stateful graph execution
- Simple terminal chat interface
- Beginner-friendly project structure

---

## 🛠️ Technologies Used

- Python
- LangGraph
- LangChain
- Groq API

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/MeghaNadh-blip/LangGraph-Projects.git
```

Move into project folder:

```bash
cd LangGraph-Projects
```

Create virtual environment:

```bash
python3 -m venv venv
```

Activate virtual environment:

### Mac/Linux

```bash
source venv/bin/activate
```

Install dependencies:

```bash
pip install langgraph langchain langchain-groq
```

---

## ▶️ Run the Project

```bash
python3 app.py
```
